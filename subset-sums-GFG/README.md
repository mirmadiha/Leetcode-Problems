# [Subset Sums](https://www.geeksforgeeks.org/problems/subset-sums2234/1)

## Problem Statement
Given an integer array `arr`, return the sums of all subsets in a list. Return the sums in any order.

## Examples
Input: arr[] = [2, 3]  
Output: [0, 2, 3, 5]

Input: arr[] = [1, 2, 1]  
Output: [0, 1, 1, 2, 2, 3, 3, 4]

Input: arr[] = [5, 6, 7]  
Output: [0, 5, 6, 7, 11, 12, 13, 18]

## Constraints
1 ≤ arr.size() ≤ 15  
0 ≤ arr[i] ≤ 10^4
