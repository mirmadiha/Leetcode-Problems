# Linked List End Insertion

## Problem Statement

You are given the head of a Singly Linked List and a value `x`. Insert the value `x` at the end of the Linked List and return the head of the modified Linked List.

## Examples

### Example 1
**Input:**  
x = 6  

**Output:**  
1 -> 2 -> 3 -> 4 -> 5 -> 6  

**Explanation:**  
6 is inserted at the end of the linked list.

### Example 2
**Input:**  
x = 1  

**Output:**  
4 -> 5 -> 1  

**Explanation:**  
1 is inserted at the end of the linked list.

## Constraints

- 0 ≤ number of nodes ≤ 10⁵  
- 0 ≤ node->data, x ≤ 10³
