# Deletion at the End of a Linked List

## Problem Statement

Given a singly linked list, delete the last node (tail) of the linked list and return the new head.

## Examples

**Input:**

```
1 -> 2 -> 3 -> 4 -> 5
```

**Output:**

```
1 -> 2 -> 3 -> 4
```

**Input:**

```
3 -> 12 -> 8
```

**Output:**

```
3 -> 12
```

## Constraints

* 1 ≤ number of nodes ≤ 10^5
* 1 ≤ node->data ≤ 10^5
