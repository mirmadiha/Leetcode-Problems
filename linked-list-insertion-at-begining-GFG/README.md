# Linked List Insertion At Beginning

## Problem Statement

You are given the head of a Singly Linked List and a value `x`. Insert the value `x` at the beginning of the Linked List and return the head of the modified Linked List.

## Examples

### Example 1
**Input:**  
x = 1  

**Output:**  
1 -> 2 -> 10  

**Explanation:**  
1 is inserted at the beginning of the linked list.

### Example 2
**Input:**  
x = 1  

**Output:**  
1 -> 2 -> 3 -> 4 -> 5  

**Explanation:**  
1 is inserted at the beginning of the linked list.

## Constraints

- 1 ≤ number of nodes ≤ 10⁵  
- 1 ≤ node->data, x ≤ 10³
